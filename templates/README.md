# Phishing Website Detector

## Project Description
Phishing Website Detector is a cybersecurity mini project that checks whether a website URL is safe or suspicious.

## Technologies Used
- Python
- Flask
- HTML
- CSS
- Git and GitHub

## Features
- Checks whether URL uses HTTPS
- Detects suspicious keywords
- Detects long URLs
- Detects IP address-based URLs
- Detects unusual domain patterns
- Displays result with reasons
